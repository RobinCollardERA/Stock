Scanner avec la caméra

Alignez la ligne rouge sur le code-barres à lire — seul le code sur la ligne est pris en compte. Le bon code : uniquement des chiffres, 13 (ou 8) chiffres imprimés sous les barres, souvent près de la mention « EAN ». Évitez S/N, MAC, IP, REF. Netteté variable ? Changez d'objectif avec le bouton 🔄.
